# Zetu — WhatsApp API for African Businesses

**Powered by Zetu & Kadem**

A self-hosted WhatsApp HTTP API for sending and receiving messages, managing sessions, webhooks, and automating workflows — built for African businesses.

## Quick Start

- **Dashboard:** https://your-railway-url.up.railway.app
- **API Docs:** https://your-railway-url.up.railway.app/swagger
- **API Port:** 2785

## Features

- ✅ Multi-session WhatsApp support
- ✅ REST API for sending messages (text, image, video, audio, documents)
- ✅ Webhook support for receiving messages
- ✅ Web Dashboard
- ✅ SQLite / PostgreSQL support
- ✅ API Key authentication
