# Community Integrations

Third-party integrations and adapters built by the community on top of Zetu's REST API.

> ⚠️ **These projects are community-maintained and are not affiliated with or endorsed by Zetu.**
> They may lag behind Zetu API changes. Verify compatibility and review the source before using
> them in production.

| Integration | Platform | Source | Notes |
| --- | --- | --- | --- |
| **ioBroker.openwa** | [ioBroker](https://www.iobroker.net/) (home / IoT automation) | [ThorstenBoettler/ioBroker.openwa](https://github.com/ThorstenBoettler/ioBroker.openwa) · [npm](https://www.npmjs.com/package/iobroker.openwa) | Send text / image / video / audio / document messages to chats and groups through Zetu's REST API, with Blockly blocks. Early stage. |

Maintain an integration built on Zetu and want it listed here? Open a PR adding a row (or an issue), including the repository link and a one-line description.

For the first-party n8n nodes, see [n8n Integration](./22-n8n-integration.md).
